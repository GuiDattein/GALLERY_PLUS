# 📸 Gallery Plus

Projeto final da **Formação React** da Rocketseat 🚀

Uma aplicação completa de galeria de fotos e álbuns, feita para praticar e consolidar todos os conceitos aprendidos na formação.

<img width="1010" height="535" alt="image" src="https://github.com/user-attachments/assets/c993c64d-7206-478e-8bc5-ff15efb6f160" />

## ✨ Funcionalidades

- Componentização dinâmica e reutilizável
- Gerenciamento de estado (useState, useEffect, useTransition)
- Navegação entre páginas com React Router
- Busca e filtros em tempo real
- CRUD completo de fotos e álbuns:
  - Criação de novos itens
  - Edição
  - Exclusão
  - Listagem
- Design com Tailwind CSS
- Tipagem forte com TypeScript
- Validação de formulário

## 🛠️ Tecnologias / Stack

- **Frontend**: ReactJS + Vite
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS
- **Roteamento**: React Router DOM
- **Gerenciamento de estado**: Context API
- **Outras libs**: react-router, react-hook-form, zod, axios, sonner

## 🚀 Como rodar o projeto localmente

### Pré-requisitos

- Node.js 18+
- npm ou yarn ou pnpm

### Passo a passo

1. Clone o repositório ou baixe os arquivos
```bash
git clone https://github.com/GuiDattein/GALLERY_PLUS.git
```
2. Entre na pasta do projeto
```
cd GALLERY_PLUS
```

⚠️ Adicione o arquivo `.env` como no exemplo `.env.example`

3. Instale as dependências
```
npm install
# ou
yarn install
# ou
pnpm install
```
4. Inicie o servidor back-end
```
npm run dev-server
# ou
yarn dev-server
# ou
pnpm dev-server
```
5. Inicie o servidor front-end
```
npm run dev
# ou
yarn dev
# ou
pnpm dev
```
